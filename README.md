# QianQian AI (QianQian AI Assistant)

This is a convenient plugin designed for Obsidian that integrates a powerful AI assistant directly into your sidebar. It replicates the core functionality of the Siyuan Note plugin of the same name, allowing you to seamlessly use the world's top AI models, workflows, and tools via the sidebar without leaving your note-taking interface.

## ðŸš€ How to Use

1. **Open Assistant**: After installing and enabling the plugin, click the red **Heart â¤ï¸ Icon** in the Obsidian left ribbon.
    
2. **Switch Models**: A dropdown menu will appear at the top of the sidebar. Click it to select the AI model or tool you want to use.
    
3. **Unlock Premium**:
    
    - Go to `Settings` -> `Community plugins` -> `QianQian AI`.
        
    - Click the "Enter Activation Code" button (or directly toggle the Premium Member switch) to unlock more powerful AI websites.
        
4. **Adjust View**: You can adjust the width of the AI window just like dragging a normal Obsidian sidebar.

### Manual Installation

1. Download the `main.js`, `manifest.json`, and `styles.css` files from the latest Release.
    
2. Place the files into your vault directory at `.obsidian/plugins/QianQian-ai-sidebar/`.
    
3. Restart Obsidian and enable the plugin.
